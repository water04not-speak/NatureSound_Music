# Changelog

All notable changes to this project will be documented in this file.

## v0.1.0 - Public open source baseline

Initial public open source baseline after copyright-sensitive resource cleanup.

### Added

- Public demo data structure for music playback and lyric synchronization.
- Placeholder resource guidance for local legally licensed assets.
- Documentation for architecture, repository readiness, screenshots, and resource boundaries.
- Open source project infrastructure: license, contribution guide, security policy, code of conduct, roadmap, changelog, and GitHub collaboration templates.

### Changed

- Project positioning updated toward a HarmonyOS / ArkTS / ArkUI music app learning project and engineering reference template.
- README now clarifies copyright and resource boundaries for public collaboration.

### Removed

- Public repository distribution of complete copyright-protected audio, complete commercial lyrics, and real music resource links.
