# Roadmap

This roadmap keeps the project focused on open source learning value, HarmonyOS engineering reference value, and safe non-heritage music content organization.

## Near Term

- Improve lyric synchronization stability across play, pause, seek, and song switching.
- Add focused tests for playback state transitions and lyric parsing.
- Continue permission minimization, especially media read/write permission review.
- Refine README setup steps, resource boundaries, and troubleshooting notes.
- Establish release management with changelog entries and GitHub Releases.

## Medium Term

- Expand Hypium unit tests for utilities, repositories, and pure playback state logic.
- Improve dark mode consistency.
- Strengthen tablet and 2in1 layout adaptation.
- Add basic accessibility checks for contrast, labels, and navigation.

## Long Term

- Document a safe way to connect legally licensed local or remote music resources.
- Explore optional backend synchronization for user data, playlists, comments, and history.
- Improve recommendation logic while keeping demo data legally clean and reproducible.
